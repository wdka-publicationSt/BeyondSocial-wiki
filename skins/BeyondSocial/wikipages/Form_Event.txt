=About=
This is the form page that can be used to add an '''event'''. The output of this form is placed in the [[Template:Event]].

The form contains fields for the following elements: 
 
 Location
 Date
 Time
 Event type
 Image
 Image caption
 
 About
 Internal (wiki) links
 External links (webpages)
 
 Documentation
